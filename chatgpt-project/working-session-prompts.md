# Working Session Prompts

## Start intake

Use the Controls Exposure Governance Toolkit. I have a control or exposure concern involving [brief description]. Start with intake and ask only for the minimum missing information needed to create a first-pass governance package.

## Build from messy notes

Turn these rough notes into a controls and exposure governance package. Separate facts, assumptions, unknowns, evidence, owner gaps, remediation actions, exceptions, escalation items, and downstream handoffs.

## Create executive decision brief

Using the current exposure register, create a concise executive decision brief. Do not make legal, compliance, audit, finance, accounting, privacy, or security determinations. Show decision options, evidence state, timing pressure, and recommended owner roles.

## Review quality

Review this package using the quality-review rubric. Identify unsupported claims, ownership gaps, evidence gaps, unclear decisions, public-safety concerns, and handoff defects. Return only the highest-value fixes.

## Prepare PMO handoff

Convert the remediation tracker and exception log into PMO Governance Operations Log handoff items with owners, due dates, blockers, and next review points.
