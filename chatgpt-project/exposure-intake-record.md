# Exposure Intake Record

Use this record to capture the minimum facts needed to build a first-pass governance view.

## Intake fields

| Field | Guidance |
|---|---|
| Intake ID | Use EXP-001 style numbering. |
| Source | Note, meeting, audit finding, supplier report, billing review, acquisition review, UAT finding, etc. |
| Concern summary | Plain-language description. |
| Affected workflow | Billing, reimbursement, supplier management, access, release, data, evidence, etc. |
| Exposure type | Financial, compliance ambiguity, audit evidence gap, supplier, process control, etc. |
| Materiality signal | Dollar range, customer/partner impact, recurrence, aging, audit interest, executive visibility, or unknown. |
| Current control state | Known, unclear, weak, missing, inconsistent, not assessed, or unknown. |
| Evidence state | Available, partial, conflicting, missing, not yet requested, or unknown. |
| Known owners | Process, control, remediation, evidence, decision, sponsor. |
| Timing pressure | Audit date, launch date, close date, supplier deadline, executive forum, or none known. |
| Immediate containment question | What must be paused, reviewed, isolated, validated, or monitored by accountable humans? |
| Assumptions | Mark unconfirmed assumptions clearly. |
| Required human review | Name roles, not invented people. |

## Intake quality rule

Do not over-question. If the user provides enough to draft a working package, proceed and mark gaps.
