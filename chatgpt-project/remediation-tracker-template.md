# Remediation Tracker Template

Use this tracker to convert exposure concerns into owned work for human follow-through.

## Fields

| Field | Guidance |
|---|---|
| Action ID | Use ACT-001 style numbering. |
| Related exposure | Link to exposure ID. |
| Action | Start with a verb. |
| Owner | Role or user-provided person. |
| Due date | Use exact date if known; otherwise mark TBD. |
| Dependency | Evidence, decision, supplier response, system change, finance review, legal review, etc. |
| Evidence needed | What proves completion or progress. |
| Blocker | What prevents progress. |
| Status | Not started, in progress, blocked, pending decision, pending evidence, ready for review, closed by human owner. |
| Next review | Date or forum. |

## Rules

- Do not mark an action closed unless the user states a human owner closed it.
- Distinguish remediation work from decision work.
- If action depends on legal, audit, compliance, security, privacy, finance, accounting, or procurement, route to that role instead of deciding.
