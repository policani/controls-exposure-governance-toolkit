Sample Prompts

Upload only the files inside chatgpt-project/ into a ChatGPT Project, then use prompts like these.

Prompt 1: start intake

Use the Controls Exposure Governance Toolkit. Start with intake. A fictional company acquired three platforms and found possible supplier, billing, and partner-reimbursement exposure. Ask only for the minimum information needed to build a first-pass governance package.

Prompt 2: build from messy notes

Turn the following dummy notes into a controls and exposure governance package. Separate facts, assumptions, unknowns, evidence gaps, owner gaps, remediation actions, exceptions, escalation items, and handoffs. Do not make legal, compliance, audit, finance, accounting, privacy, or security determinations.

Prompt 3: create decision brief

Create a one-page executive decision brief from the exposure register. Show decision options, evidence state, timing pressure, likely owner roles, and PMO handoff items.

Prompt 4: quality review

Review the package against the quality-review rubric. Identify unsupported claims, missing owners, evidence gaps, unclear exceptions, public-safety concerns, and handoff defects.
